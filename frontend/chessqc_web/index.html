CHESS-QC

Coastal Hazards, Engineering, and Structures System (CHESS) — Quick Compute (QC)

35+ coastal-engineering calculators covering wave prediction and theory, transformation, structural design, runup and overtopping, littoral processes, inlets, harbors, and storm surge. They run in a web browser with no install.

A reimplementation of the legacy USACE ACES suite and other methods for back-of-the-envelope calculations.

Launch calculators → Documentation

Open source

Every application is a single, dependency-light Python module that runs its own validation and prints a worked example.

SI & US units

Switch unit systems per calculator; values convert at the edge while the physics runs in SI internally.

Validated

Each tool is checked against the ACES User's Guide worked examples or analytic limits, with a fidelity label: 28 exact, 3 standard, 5 provisional.

Serverless

Runs entirely in the browser via Pyodide, with no backend and no install. A native desktop app and command-line front-end are available from the GitHub repository.
